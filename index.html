高校期末成绩查询系统

安全 · 便捷 · 高效的成绩发布与查询平台

基于 GitHub Pages 静态托管
📚

管理员发布成绩

上传平时成绩和期末成绩的 Excel 文件，输入课程名称，一键生成可分享的查询链接
🔎

学生查询成绩

输入姓名和学号即可安全查询个人成绩，包含平时成绩、期末成绩及详细分项
📝 使用说明
1
管理员打开「管理员发布成绩」页面，上传从教务系统导出的平时成绩和期末成绩 Excel (.xlsx) 文件。
2
输入课程名称（如"高等数学"），设置平时/期末成绩的占比权重，点击「生成并发布」。
3
系统会生成一个 JSON 数据文件。管理员将文件上传到 GitHub 仓库的 data/ 目录下，并通过 GitHub Pages 部署。
4
系统自动生成查询链接。将链接分享给学生，学生输入姓名 + 学号即可查询自己的成绩。
🔒 安全说明

本系统采用纯前端静态方案，所有数据以 JSON 格式存储在 GitHub 仓库中。 学生查询时需要同时输入正确的姓名和学号才能查看成绩，其他同学的成绩信息不会被泄露。 数据在浏览器端进行匹配，不会发送到任何第三方服务器。
